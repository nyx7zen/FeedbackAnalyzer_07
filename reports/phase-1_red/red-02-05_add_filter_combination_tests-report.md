## 필터 조합 분석 현황

### 지원되는 필터 조합
- ✅ 배송 + 품질: 모두 카운팅
- ✅ 배송만: 배송만 카운팅
- ✅ 품질만: 품질만 카운팅
- ✅ 미매칭: 모든 카운트 0

### 카테고리별 키워드 예시
- 배송: "배송", "택배", "도착", "출고", "지연"
- 품질: "품질", "제품", "하자", "불량", "내구성"
